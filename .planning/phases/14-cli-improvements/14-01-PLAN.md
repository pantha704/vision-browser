# Phase 14, Plan 01: CLI Improvements

## Goal
Add real-time progress indicators, readable error messages, task summary, and graceful Rich fallback.

## Requirements
- CLI-01: Real-time progress indicators during task execution
- CLI-02: Human-readable error messages with suggested fixes
- CLI-03: Task summary report on completion
- CLI-04: Rich is optional -- graceful fallback to basic output when unavailable

## Implementation

### 1. CLI-04: Graceful Rich Fallback
```python
try:
    from rich.console import Console
    from rich.panel import Panel
    HAS_RICH = True
except ImportError:
    HAS_RICH = False

def get_console():
    if HAS_RICH:
        return Console()
    return None  # Use print fallback
```

### 2. CLI-01: Progress Indicators in FastOrchestrator
- Already has some Rich console.print statements
- Add a `TaskProgress` context manager/class that tracks:
  - Current turn, total turns
  - Current action being performed
  - Time elapsed
- Display as: `[Turn 3/20] → Clicking element 5... (elapsed: 12s)`

### 3. CLI-02: Readable Error Messages
- Map exception types to user-friendly messages with suggestions:
  - BrowserNotInstalledError: "Browser not found. Install Chromium: playwright install chromium"
  - ConfigError: "Configuration error: {detail}. Check config.yaml or environment variables."
  - VisionAPIError: "Vision model error: {detail}. Check NVIDIA_API_KEY is set and valid."
  - TimeoutError: "Operation timed out. Check network connection or increase timeout."

### 4. CLI-03: Task Summary
- On completion, display:
  ```
  ── Task Summary ──────────────────────────
  Task: Go to google.com and search for cats
  Status: ✅ Complete
  Turns: 5
  Actions: 8 (7 succeeded, 1 failed)
  Time: 23.4s
  Final URL: https://www.google.com/search?q=cats
  ──────────────────────────────────────────
  ```

### 5. Track metrics in FastOrchestrator
- `_total_actions`, `_succeeded_actions`, `_failed_actions`
- `_start_time` for elapsed time
- `_final_url` on completion

## Verification
- CLI works with Rich installed
- CLI works without Rich (mock ImportError)
- Error messages are readable and include suggestions
- Task summary displays correct metrics
- All existing CLI commands work identically
